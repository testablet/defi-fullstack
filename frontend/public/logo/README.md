# Logo du Projet

Placez votre logo ici dans les formats suivants :
- `logo.png` ou `logo.svg` (recommandé)
- `logo-dark.png` ou `logo-dark.svg` (pour les variantes sombres)
- `logo-icon.png` ou `logo-icon.svg` (icône seule)

Le logo sera automatiquement chargé dans l'application.

Formats supportés : PNG, SVG, JPG
Taille recommandée : 200x60px pour le logo complet, 40x40px pour l'icône
