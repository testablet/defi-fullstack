# Changelog

## [1.0.0] - 2025-01-01

### Added
- Backend Symfony 7.1 avec architecture DDD
- Frontend Vue.js 3 + TypeScript + Vuetify 3
- API REST conforme OpenAPI 3.1
- Algorithme de routage Dijkstra
- Statistiques agrégées par code analytique
- Authentification JWT
- Tests unitaires et d'intégration (≥80% couverture)
- CI/CD GitHub Actions complet
- Docker Compose pour déploiement
- Documentation complète

### Features
- Calcul de trajet entre deux stations
- Visualisation des statistiques avec graphiques
- Interface utilisateur moderne et responsive
- Persistance des trajets en base de données

### Security
- Authentification JWT
- Validation des entrées
- Headers de sécurité
- Scan de vulnérabilités dans CI/CD
